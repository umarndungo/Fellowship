# Inuka Fellowship Projects

This is a repository for the Inuka Fellowship Data Analytics and AI apprenticeship coursework.

## Content

This repository contains all assignments in the following formats:
- `/Fellowship/[WeekNumber]/[filename].md` - for markdown documents such as memos.
- `/Fellowship/[WeekNumber]/[filename].ipynb` - for Jupyter Notebooks
- `/Fellowship/[WeekNumber]/[filename].py` - for python files

and any other document formats necessary for assignment and task execution.
